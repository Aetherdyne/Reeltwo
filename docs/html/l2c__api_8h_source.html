154 /* Congestion status callback protype. This callback is optional. If
155 ** an application tries to send data when the transmit queue is full,
156 ** the data will anyways be dropped. The parameter is:
157 ** Local CID

163 /* Transmit complete callback protype. This callback is optional. If
164 ** set, L2CAP will call it when packets are sent or flushed. If the
165 ** count is 0xFFFF, it means all packets are sent for that CID (eRTM
166 ** mode only). The parameters are:
167 ** Local CID
168 ** Number of SDUs sent or dropped
169 */
170 typedef void (tL2CA_TX_COMPLETE_CB) (UINT16, UINT16);

229 ** services.

252 /*******************************************************************************
253 **
254 ** Function L2CA_ErtmConnectReq
255 **
256 ** Description Higher layers call this function to create an L2CAP connection
257 ** that needs to use Enhanced Retransmission Mode.
258 ** Note that the connection is not established at this time, but
